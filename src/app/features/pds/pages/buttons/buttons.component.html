 Guía de Uso

Primary

Acciones principales como guardar, enviar, confirmar. Use color="primary"

Secondary

Acciones secundarias o alternativas. Use class="btn-secondary"

Accent

Acciones destacadas o llamados a la acción. Use color="accent"

Error/Warn

Acciones destructivas como eliminar, cancelar. Use color="warn"

Tertiary

Acciones terciarias o de bajo énfasis. Use class="btn-tertiary"

 Ejemplo de Código 
// Botón Primary Raised con tamaño medium y forma square
<button mat-raised-button color="primary" class="btn-square btn-medium">
  Primary
</button>

// Botón Secondary con icono
<button mat-raised-button class="btn-secondary btn-square btn-medium">
  <mat-icon>add</mat-icon>
  Add Item
</button>

// Botón Tertiary Outlined
<button mat-stroked-button class="btn-tertiary btn-rounded btn-small">
  Tertiary
</button>
